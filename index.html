How to Add More Images

  1. Find an existing image card in the HTML code (look for the section with class "image-card")
  2. Copy the entire block of code for one image card
  3. Paste it right after the existing image cards (inside the <div class="gallery"> section)
  4. Change the image-title and image-url to your new image details
  5. Save the file and refresh your browser to see the new image

See the template section below for a copy-paste ready code block.

Copy-Paste Template

Copy this code block and paste it inside the <div class="gallery"> section to add a new image:

<!-- Image 5 -->
<div class="image-card">
  <div class="image-container">
    <img src="PASTE-IMAGE-URL-HERE" alt="IMAGE TITLE HERE">
  </div>
  <div class="image-info">
    <h3 class="image-title">IMAGE TITLE HERE</h3>
    <p class="image-url">PASTE-IMAGE-URL-HERE</p>
  </div>
</div>

Instructions: Replace "IMAGE TITLE HERE" with your image title and "PASTE-IMAGE-URL-HERE" with the direct URL to your image.
